Sample questions

1. A 30-year-old patient presents with hypertension. Which of the following is a selective beta-1 blocker?
a. Propranolol
b. Atenolol
c. Labetalol
d. Carvedilol

2. Which drug is the first-line treatment for type 2 diabetes mellitus?
a. Glibenclamide
b. Insulin glargine
c. Metformin
d. Sitagliptin

3. A patient on warfarin therapy requires which monitoring parameter?
a. aPTT
b. Platelet count
c. PT/INR
d. Bleeding time

Q4. Which of the following is a loop diuretic?
a. Hydrochlorothiazide
b. Spironolactone
c. Furosemide
d. Acetazolamide

5) Which neurotransmitter is deficient in Parkinson's disease?
a. Acetylcholine
b. Dopamine
c. Serotonin
d. GABA

Mixed formats within the same file are accepted. The tool will detect the dominant pattern and flag any outliers in the console.
